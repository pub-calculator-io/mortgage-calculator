Calculate your monthly payment, total property ownership costs, and amortization timeline with options for taxes, PMI, HOA, and early payments with our free mortgage calculator.

[https://www.calculator.io/mortgage-calculator/](https://www.calculator.io/mortgage-calculator/)

== Usage ==

Add the Mortgage Calculator shortcode to your page, post or sidebar:

`[ci_mortgage_calculator]`

Add the following code to your template where you would like the Mortgage Calculator to appear:

`<?php display_ci_mortgage_calculator(); ?>`

== Installation ==

1. Upload the Mortgage Calculator /ci_mortgage_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Mortgage Calculator plugin through the "Plugins" menu in WordPress.
